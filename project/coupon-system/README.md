##요구사항 정의

선착순 100명에게 할인 쿠폰을 제공하는 이벤트를 진행하고자 합니다.

이 이벤트는 아래와 같은 조건을 만족해야 한다.
- 선착순 100명에게만 지급되어야 한다.
- 101개 이상이 지급되면 안된다.
- 순간적으로 몰리는 트래픽을 버틸 수 있어야 합니다.
